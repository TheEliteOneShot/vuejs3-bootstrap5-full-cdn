# vuejs3-bootstrap5-full-cdn
VueJS3 using Bootstrap 5, Material Design, As Grid Enterprise, Vue Toast Notification, and Box Icons. All the scripts have been included, no dependencies or install process required. Simply run this in a webserver and serve the index.html.
